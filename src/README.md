Frontend for Constel social media network.

FEATURES:
Log in
Post (supports text and audio),
Like post,
Play audio (if audio post)
Delete post,
Comment,
Delete comment
Log out

During audio recording, audio is visualized on the canvas.

TECNOLOGIES USED:
React, typescript, tailwind css, Web Audio API.
